
wowTeamComp is a project I created to help new World of Warcraft players

1- My project offers the user the perfect team for the in-game competitions,
using the API of raider.io, a platform that holds all the information related 
to the in-game competitions.

2-(in developement) The application also offers the possibility to create an account and save preferred teams.

Used packages:
-Gorilla mux

Used frameworks:
-Gin gonic
-React
